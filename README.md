# ⚡ EV Novated Lease Calculator — NSW

A beautiful, interactive calculator to determine whether novated leasing an electric vehicle in NSW is financially worth it.

🔗 **Live Demo:** https://schoi0209.github.io/EV-Novated-Lease-Calculator/

---

## Features

- **FBT Exemption** — Auto-detects eligible EVs under the $89,332 LCT threshold (FY2024–25)
- **Full Tax Calculation** — ATO 2024–25 income tax rates + 2% Medicare levy
- **GST Saving** — Employer GST claim benefit automatically applied
- **Running Costs** — Pre-tax packaging of rego, insurance, servicing, electricity, admin
- **Side-by-side Comparison** — Novated lease vs standard car loan
- **Verdict Engine** — Clear 🚀 / ✅ / 🤔 / ❌ recommendation
- **8 EV Presets** — Tesla, BYD, Hyundai, Kia, Polestar, Mercedes, BMW
- **🌀 Tubes Cursor Effect** — Interactive 3D animated cursor by [Kevin Levron](https://codepen.io/soju22/pen/qEbdVjK) (CC BY-NC-SA 4.0)
- **Light / Dark Mode** — System theme detection + manual toggle
